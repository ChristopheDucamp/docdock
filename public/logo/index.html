Emplacement du Logo

Créez une page _header.md dans le répertoire content. Son contenu est ce que vous obtiendrez à l’emplacement du logo (tout en haut à gauche de l’écrant).

Considération pour l’emplacement du logo
Ne réglez pas [menu.main] dans le frontmatter de votre fichier _header.md
Truc : vous pouvez ajouter une image, une combobox avec des liens vers une autre documentation…
Truc 2 : regardez Entrées menu supplémentaires si vous voulez ajouter des liens vers un autre site web dans cette barre latérale
Christophe Ducamp 01/05/2017
