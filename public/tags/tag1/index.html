Créer une Page

Hugo-theme-docdock définit deux types de pages. Default et Slide. Default est la page commune comme celle que vous êtes en train de lire. Slide est une page qui utilise le plein écran pour afficher son contenu en markdown comme une présentation reveals.js. HomePage est un contenu spécial qui s’affichera comme un contenu de page d’accueil. Pour dire à Hugo-theme-docdock de considérer une page comme un diapo, ajoutez simplement dedans un type="slide" dans le frontmatter de votre fichier. »
