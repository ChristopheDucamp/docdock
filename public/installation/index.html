Installation

Les étapes qui suivent sont là pour vous aider à initialiser votre nouveau site web. Si vous ne connaissez pas du tout Hugo, nous vous suggérons de vous entraîner en suivant ici les instructions de paramétrage ou cette documentation rapide pour débutants.

Créer votre Documentation

Hugo fournit une commande new pour créer un nouveau site web.

$ hugo new site <nouveau_site_web>

Installer le Thème

Installez le thème Hugo-theme-docdock en suivant ceci

Déplacez-vous dans le répertoire thème et téléchargez le thème

$ cd themes
$ git clone https://github.com/vjeantet/hugo-theme-docdock.git docdock

Alternativement, vous pouvez télécharger le thème sous fichier .zip et l’extraire dans le répertoire themes.
